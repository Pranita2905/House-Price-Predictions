# 🏠 House Price Prediction System

A Machine Learning-powered web application that predicts house prices based on property features such as bedrooms, bathrooms, house size, lot size, and ZIP code. The application is built using Flask and a K-Nearest Neighbors (KNN) Regression model.

# 🚀 Live Application:
🔗 [Try the Live Application](https://house-price-prediction-c1fz.onrender.com)

Click the link to try the application.
---

## 📌 Project Overview
The House Price Prediction System helps estimate property prices by analyzing key house characteristics. This project demonstrates the complete Machine Learning workflow, including:
* Data Cleaning
* Data Preprocessing
* Outlier Removal
* Feature Selection
* Model Training using KNN Regression
* Model Serialization using Pickle
* Web Deployment using Flask
